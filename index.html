Two-Tier Hybrid Architecture

ivygrep uses a dual-engine architecture: fast indexing with progressive quality upgrades.

1

Hash Index

Sub-second indexing via 128-bit SIMD structural hashes. Instant search availability.

→
2

Hybrid Fusion

Lexical BM25 (Tantivy) and Semantic paths fuse via RRF to return high-precision results.

→
3

Neural Enhancement

Background daemon silently upgrades chunks to quantized ONNX (AllMiniLML6V2Q) with Apple CoreML acceleration on macOS.

Speed That Scales

Supercharge Your AI Agents

ivygrep exposes an MCP (Model Context Protocol) server. Instead of stuffing entire files into your LLM context, your agent can query your codebase natively.

Claude Code

claude mcp add -s user ig -- ig --mcp

Cursor IDE

"ig": { "command": "ig", "args": ["--mcp"] }

OpenCode
